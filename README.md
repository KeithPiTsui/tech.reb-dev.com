# tech.reb-dev.com

A simple cms server for [tech.reb-dev.com](https://tech.reb-dev.com)

## Prerequisites

#### Swift 
- 3.0.2

#### macOS
- macOS Sierra 10.12.2

#### Linux
- Ubuntu 14.04

#### MySQL

```bash
$ sudo apt-get install mysql-server libmysqlclient-dev
```

#### Discount

```bash
$ sudo apt-get install libmarkdown2-dev
```
