# vapor-cms

A simple cms server written by swift.

## Prerequisites

#### Swift 
- 3.0 Release

#### macOS
- OS X El Captain 10.11.6

#### Linux
- Ubuntu 14.04

#### MySQL

```bash
$ sudo apt-get install mysql-server libmysqlclient-dev
```

#### Discount

```bash
$ sudo apt-get install libmarkdown2-dev
```
